This is a basic Flask web application that showcases authentication and processing for users and forms as well as data manipulation through the use of Python functions.

# Features
Features
- Login and logout functionality with role-based access (Admin/Student) for users
- Greet page with string manipulation operations (uppercase, lowercase, reverse, length)
- Favorites: Favorites page to manage programming languages
- Admin-only actions (delete multiple languages, clear list)

- Session-based Authentication
