# Flask mini app

A basic Flask web application supporting login functionality, sessions, and form processing.

# Features
- Login & Logout System (Admin/Student)
- Greeting page with string operation
- Favourites page for handling programming languages

- Delete and Clear Data controls restricted to Administrators
